# Changelog

## v0.2.0 (2026-06-29)

### Neu
- **App-Filter mit drei Modi**: Aus / Blacklist / Whitelist
  - **Aus**: Overlay immer sichtbar (Standard)
  - **Blacklist**: Overlay in gelisteten Apps ausblenden
  - **Whitelist**: Overlay nur in gelisteten Apps anzeigen
- Settings UI für App-Filter komplett überarbeitet (deutsche Labels)

### Behoben
- OverlayService pollt nicht mehr unnötig SharedPreferences bei leerer Blacklist

## v0.1.0 (2026-06-28)

### Neu
- Overlay mit Echtzeit-Systeminfo (CPU, RAM, Batterie, Speicher, Wetter)
- 6 integrierte Themes (Minimal, CPU Monitor, Badazz89, Neon Punx, Inferno, Neon)
- Theme-Editor mit Live-Vorschau und Drag-and-Drop-Sortierung
- Theme-Import/Export als JSON
- Free-Move-Modus (komplette Widget-Gruppe verschieben)
- Batteriesparmodus
- Konfigurierbare Transparenz, Aktualisierungsrate und Schriftart
- Multi-Language-Unterstützung (DE/EN)
